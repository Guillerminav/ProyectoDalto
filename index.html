¿Qué es el Desarrollo Web?

El Desarrollo Web es la construcción y el mantenimiento de los sitios web y aplicaciones.

Diferencias entre Diseño Web y Programación Web

El diseño web define estructuras funcionales para el usuario, lo que logra a través del diseño de distintos elementos buscando la mejor experiencia de usuario y la optimización de los procesos. En cuanto a la programación web, comprende aspectos funcionales con un enfoque práctico, materializando la base estructural del proyecto. Por ello, podemos decir que programación web y diseño web van de la mano, pero con funciones diferenciadas y sobre todo puntos de vista diferentes.

Desafíos para el primer mes

  • Estructurar correctamente una web.
  • SEO básico.
  • Armar formulario de contacto estático responsive.
  • Armar presentación de algun tema que te apasione, algo interesante.
  • Armar una web contando quienes son y qué hacen.

Desafíos extra

  • Armar un reloj de agujas con solo HTML y CSS.
  • Crear un corazón que palpita.
  • Crear un efecto de glitch en alguna imagen.
